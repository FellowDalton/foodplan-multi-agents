## Typography

- **Page Title**: 3xl, Bold, Gray-900
- **Section Headers**: lg, Semibold, Gray-900
- **Product Names**: lg, Semibold, Gray-900
- **Prices**: 3xl, Bold, Gray-900
- **Price per Unit**: sm, Medium, Gray-700
- **Badges**: xs, Medium
- **Stats**: 2xl, Bold, Gray-900

## Spacing

- Card padding: 1rem (16px)
- Grid gap: 1.5rem (24px)
- Section margin: 1.5rem (24px)
- Button padding: 0.5rem 1rem (8px 16px)

## Shadows

- Cards: `shadow-md` (medium shadow)
- Cards hover: `shadow-xl` (extra large shadow)
- Tabs: `shadow-lg` (large shadow)

## Transitions

- All hover states: 200ms ease
- Shadow transitions: 200ms
- Color transitions: 200ms

- **Scannable**: Easy to browse while shopping
- **Informative**: All key info at a glance
- **Actionable**: One-click to add to list
- **Beautiful**: Clean, modern design
- **Accessible**: Good contrast, clear labels
- **Fast**: Instant filtering and sorting
